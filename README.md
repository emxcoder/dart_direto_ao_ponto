# ---

**🎯 Dart Direto ao Ponto**

Este repositório contém todo o código-fonte, exercícios e materiais de apoio da playlist **"Dart Direto ao Ponto"** no YouTube. O objetivo deste curso é ensinar a linguagem Dart de forma objetiva, focando no que realmente importa para o mercado e para o desenvolvimento com Flutter.

📺 [**Assistir à Playlist no YouTube**](https://youtu.be/6EqheOOCtFg)

## ---

**📖 Sobre o Curso**

O Dart é uma linguagem otimizada para o cliente, desenvolvida pelo Google para criar aplicativos rápidos em qualquer plataforma. Neste curso, abordamos desde a sintaxe básica até conceitos avançados de Programação Orientada a Objetos e Assincronismo.

### **O que você vai encontrar aqui:**

* Explicações claras em cada arquivo de código.  
* Desafios práticos ao final de cada aula.  
* Melhores práticas de clean code em Dart.

## ---

**🚀 Como usar este repositório**

1. **Clone o projeto:**  
   Bash  
   git clone https://github.com/SEU\_USUARIO/dart-direto-ao-ponto.git

2. **Entre na pasta do módulo desejado:**  
   Bash  
   cd modulo-01-basico

3. **Execute o código (com o Dart SDK instalado):**  
   Bash  
   dart run 1.hello\_world.dart
